## 🚀 Como Abrir o Projeto

### **Opção 1: Script Automático (Recomendado - Windows)**

Basta **duplo clique** em `start.bat`:
- Inicia um servidor local automaticamente
- Abre o Chrome na porta `8000`
- O servidor continua rodando até você fechar a janela

### **Opção 2: Com Node.js**

Se tiver Node.js instalado:

```bash
node server.js
```

O projeto abrirá automaticamente em `http://localhost:8000`

### **Opção 3: Manualmente (Qualquer navegador)**

**Windows (com Python):**
```bash
python -m http.server 8000
```

**Mac/Linux:**
```bash
python3 -m http.server 8000
```

Depois acesse no navegador: `http://localhost:8000`

## 📝 Editando o Projeto

### Adicionar/Editar Projetos
Edit o array `projects` em `main.js`

### Mudar Cores
Edit as variáveis CSS em `style.css`:
```css
:root {
    --bg-color: #121212;
    --accent-color: #bb86fc;
    /* ... */
}
```

### Atualizar Dados Pessoais
- **Email:** `index.html` (seção "Entre em Contato")
- **Telefone:** `index.html` (seção "Entre em Contato")
- **LinkedIn/GitHub:** Links no rodapé do `index.html`
